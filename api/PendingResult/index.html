PendingResult

The chain object every Controller:Shatter(...) call returns: Controller:Shatter(...):Next(Enums.Event.X):Then(callback). Listeners registered before the matching event fires run immediately when it does; listeners registered after a past event has already fired never run for that firing - each PendingResult only lives for one Shatter call.

interface Result {
Voxels: {
Whole: {Voxel},
Partial: {Voxel},
Extra: {Voxel}
}--

every produced Voxel, split three ways. Whole: isolated but never sliced (HitboxType.Whole only). Partial: the actual debris - sliced, and the only bucket Cleanup ever runs against. Extra: produced but outside the hitbox - greedy-meshed into a handful of boxes and left alone unless you Cleanup it yourself.

Instance: Instance?--

the Target/Host this Shatter call ran against. nil for indiscriminate destruction (no Target, no Host - just a Hitbox).

CollisionPoint: Vector3?--

reserved for a future raycast-driven Shatter helper; not populated today.

SurfaceNormal: Vector3?--

reserved for a future raycast-driven Shatter helper; not populated today.

}
