VoxelController

Returned by Facet.Define. Owns one Shatter pipeline: resolve candidate parts, optionally classify them against a HitboxConfig, voxelize, apply Effects, run Cleanup, and fire a Result through the PendingResult chain. Typed explicitly (never any) so every field and method below has real Intellisense.

Types​

HitboxConfig​

</>
interface HitboxConfig {
Size: Vector3?--

box/region extents, in studs, centered on CFrame. Ignored by Raycast.

CFrame: CFrame?--

where the hitbox sits in the world. Defaults to CFrame.new() (the origin) if omitted.

ViewVector: Vector3?--

cast direction for QueryType.Raycast only; ignored by every other QueryType.

Color: Color3?--

cosmetic only - tints a visualized hitbox if you render one yourself. Facet never draws it for you.

QueryType: Enums.QueryType?--

Box (default) | Sphere | Region | Raycast. Picks which exact-shape test runs after the octree broad phase.

HitboxType: Enums.HitboxType?--

Partial (default, voxelizes whatever's detected) | Whole (returns fully-contained parts untouched, no slicing).

Shape: Enum.PartType?--

reserved for a future visualized-hitbox helper; not read by Shatter today.

Copy: Instance?--

reserved for a future "clone the hitbox as a real part" helper; not read by Shatter today.

}

DefineConfig​

</>
interface DefineConfig {
Type: Enums.ControllerType--

Relative (dynamic sizing, requires RecommendedVoxelSize) or Absolute (fixed VoxelSize). Required.

RuntimeType: Enums.RuntimeType?--

Instant (default, runs every Shatter immediately) or Timeline (queues calls until :Start()).

Host: Instance?--

scopes every Shatter call on this controller to one instance without naming a Target each time.

MinimumVoxelSize: number?--

Relative only. Floor Shatter will shrink voxels to under load. Defaults to RecommendedVoxelSize (no shrink) if omitted.

RecommendedVoxelSize: number?--

Relative only. Target voxel size the controller sizes toward. Required for Relative.

Drivers: {VoxelizationMethod: string?}?--

reserved for swapping the slicing backend; not read by Shatter today.

VoxelSize: number?--

Absolute only. The fixed cube size every voxel is sliced to. Required for Absolute.

}

interface ShatterConfig {
Hitbox: HitboxConfig?--

scopes detection to a region of the target. Omit entirely to voxelize the whole target.

Effects: {[string]: any}?--

keyed by Enums.Effects (Explode = { Radius: number }, Anchored = boolean), or the string key "Velocity" for a raw Vector3.

Cleanup: (Enums.Cleanup | (Voxel) → ())?--

Rebuild | Fade | Destroy, or your own function(voxel) ... end. Only ever runs against Result.Voxels.Partial.

CleanupDuration: number?--

seconds the Cleanup tween takes (Fade/Rebuild only - Destroy is instant). Defaults to 0.5 for Fade, 0.6 for Rebuild.

}

Shatter(Config) when the controller has a [Host], Target is implied. Shatter(Target, Config) otherwise. Hitbox is optional in either form - omit it to shatter the whole Target/Host directly with no detection step.

Start​

</>
VoxelController:Start() → ()

Flushes the Timeline queue in call order. No-op for Instant controllers - they never queue, every :Shatter() call already ran by the time it returns.

Destroy​

</>
VoxelController:Destroy() → ()

Clears any queued Timeline jobs that haven't run yet. Doesn't touch parts already produced by past :Shatter() calls - those live on independently under workspace._Bin.Voxels.
